Java Project: DragonBallPixelZ

With this Java project, i wanted to aim to make a platform shooter, in which games such as Super Smash Bros, and Dragon Ball had inspired me to transfer that idea and try replicate it in the world of Java. My plan was to try and execute something similar by implementing two characters with two users being able to play. With this theme, comes different playable maps in which i tried to have a somewhat large pool in the 5 maps that can be selected by the user. There are many features included within this, such as lives, gems (dragonball) and platforms to make this game a fun and exciting experience for the users to battle it out against each other. With the java project comes a graphical user interface to make it more of a realistic game with options on the menu for the users to interact with. Ultimately, the game reflects my passions in playing these games and to attempt to make a game with such features from these brilliantly polished games.
